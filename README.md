## Featured Projects

**Smart News Summarizer**
`BART` `Hugging Face Transformers` `gTTS` `BeautifulSoup` `Flask`
A multilingual news-to-audio pipeline that pulls real-time articles via the Google News API, condenses them with a fine-tuned BART summarizer, and reads them back as speech across 65+ languages.

**OCR Translation Web App**
`Flask` `OCR.Space API` `Pillow` `Googletrans`
Pulls text out of scanned photos and PDFs with high parsing accuracy, then translates it across 50+ languages and exports the result straight to PDF, DOCX, or Excel — turning a scanned page into an editable document in one pass.

**Automated Mobile Robot**
`ROS2` `SLAM` `Nav2` `RViz2`
The navigation stack for an autonomous mobile robot, handling real-time mapping, localization, and dynamic obstacle avoidance. Tuned the path-planning pipeline to noticeably cut navigation delays over the baseline, reducing dependence on fixed infrastructure.
